Một số key trong vim giúp edit nhanh hơn

Vim hay Neovim là một editer mạnh mẽ phổ biến trên các nền tảng *nix. Bài viết này ngoài ghi chép một số vimkey cơ bản hay dùng còn note lại các vimkey được config tại vim repo trong một số plugin mà tôi hay dùng.

Một số vimkey sau đây không được sắp xếp theo các group mà chủ yếu là các vimkey tôi hay dùng. Bạn có thể xem thêm rất nhiều vimkey khác đã được group tại đây.

Một số regex hay dùng

Regex (Regular Expressions) hay biểu thức chính quy là một tập các ký tự dùng để mô tả một chuỗi nào đó theo những quy tắc, cú pháp nhất định. Regex không những được sử dụng trong lập trình mà còn kết hợp mạnh mẽ đối với các công cụ như grep, sed,… trên môi trường Linux.

Một số ký tự cơ bản trong regex:

Ký tự Ý nghĩa
\ Đươc sử dụng để bỏ qua ký tự đặc biệt
^ Ký tự đánh dấu bắt đầu 1 chuỗi
$ Ký tự đánh dấu kết thúc 1 chuỗi
. Một ký tự (đơn) bất kỳ
| sử dụng như OR. Ví dụ IPv4 | IPv6
() Sử dụng như GROUP (độ ưu tiên). Ví dụ IP (v4 | v6)
[] Một dãy các ký tự. Ví dụ [abc]. Có thể kết hợp với ^ ở đầu để chỉ phủ định (NOT) hoặc với - để chỉ ký tự liên tiếp như [0-9] (từ 0 -> 9) hay [a-z] (từ a -> z)
? Kiểm tra có 0 hoặc 1 ký tự phía trước. Ví dụ https? sẽ khớp với http và https
* Biểu diễn 0 hoặc 1 hoặc nhiều hơn 1 ký tự bất kỳ phía trước
+ Ký tự phía trước xuất hiện 1 hoặc nhiều lần
{n} Ký tự phía trước xuất hiện n lần
{min,} Ký tự phía trước xuất hiện min lần trở lên
{min,max} Ký tự phía trước xuất hiện tối thiểu min lần và tối đa max lần

Một số cái beep trong shell

Hôm nay tôi lang thang và thấy mấy cái này và tôi biết tôi cần note lại cho mình. Tôi cũng xin nói luôn tôi cũng chưa biết gọi chúng là cái gì nên tạm thời gọi là cái beep vậy. Khi làm việc với shell trên linux và… có đôi chút về bash thì nó thật sự hữu ích

Command Description Example
& Chạy lệnh với background ls &
&& Biểu thức AND if [ “$foo” -ge “0” ] && [ “$foo” -le “9”]
** Biểu thức OR if [ “$foo” -lt “0” ] ** [ “$foo” -gt “9” ]
^ Ký tự bắt đầu dòng/chuỗi grep “^foo”
$ Ký tự kết thúc dòng/chuỗi grep “foo$”
= Phép bằng hoặc gán if [ “$foo” = “bar” ]
! Phép NOT if [ “$foo” != “bar” ]
$$ PID của shell hiện tại echo “my PID = $$”
$! PID của background command cuối cùng ls & echo “PID of ls = $!”
$? Trạng thái thoát của lệnh cuối cùng ls ; echo “ls returned code $?”
$0 Tên của lệnh hiện tại echo “I am $0”
$1 Đối số đầu tiên của lệnh hiện tại echo “My first argument is $1”
$9 Đối số thứ 9 của lệnh hiện tại echo “My ninth argument is $9”
$@ Tất cả các đối số (parameters) của lệnh hiện tại echo “My arguments are $@”
$* Tất cả các đối số của lệnh hiện tại (trừ khoản trống và trích dẫn) echo “My arguments are $*”
-eq So sánh bằng if [ “$foo” -eq “9” ]
-ne So sánh không bằng if [ “$foo” -ne “9” ]
-lt So sánh nhỏ hơn if [ “$foo” -lt “9” ]
-le So sánh nhỏ hơn hoặc bằng if [ “$foo” -le “9” ]
-gt So sánh lớn hơn if [ “$foo” -gt “9” ]
-ge So sánh lớn hơn hoặc bằng if [ “$foo” -ge “9” ]
-z Chuỗi có chiều dài 0 if [ -z “$foo” ]
-n Chuỗi có chiều dài khác 0 if [ -n “$foo” ]
-d Là một thư mục if [ -d /bin ]
-f Là một file if [ -f /bin/ls ]
-r Là một file có thể đọc được if [ -r /bin/ls ]
-w Là một file có thể ghi được if [ -w /bin/ls ]
-x Là một file có thể thực thi if [ -x /bin/ls ]
