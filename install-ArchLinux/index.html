Cài đặt Arch Linux trên Virtualbox [Notes]

Arch Linux là một hệ điều hành họ Linux. Đây không phải là một hệ điều hành cho người mới bắt đầu. Bạn sẽ phải định dạng, install tất cả bằng tay khi cài đặt. Trong hướng dẫn này chúng ta sẽ xem cách cài đặt. Có thể nó sẽ không khó như bạn đã nghĩ.

1. Giới thiệu

Chúng ta sẽ xét qua một vài điểm xem như là ưu điểm của distro này sau đây.

  • Đối với Ubuntu, Fedora, Linux mint,… sau khi cài đặt chúng ta sẽ được tự động cài đặt một số phần mềm đi kèm như LibreOffice, GIMP,… Tuy nhiên với Arch, bạn sẽ không có gì và tất nhiên bạn phải tự tìm và cài đặt những thứ mình cần. Có lẽ bạn nghĩ sẽ thật phức tạp, tuy nhiên chúng ta cũng xét qua một khía cạnh khác rằng nó sẽ nhẹ hơn, nhanh hơn, tùy chỉnh hơn với người dùng.

  • Với Arch bạn sẽ có AUR (Arch user repository) - đây là kho dự trữ của Arch, nó tuyệt vời hơn bạn nghĩ. Nó cung cấp cho bạn rất nhiều các gói phần mềm không có sẵn trong kho chính thức.

  • Trình quản lý pacman tự động biên dịch các gói cho bạn.

  • Rolling release: Điều này có nghĩa là bạn sẽ luôn có những bản mới nhất, các gói, kenel, tất cả chỉ đợi bạn trải nghiệm

  • Cuối cùng giờ đây Arch cũng dần có lượng cộng đồng người dùng đông đảo, và Wiki Arch cũng hỗ trợ đầy đủ và hiện thời rất tốt.

Với Arch việc cài đặt như đã nói sẽ có chút phức tạp hơn các distro khác. Tuy nhiên cũng có những dự án nhằm đơn giản hóa cài đặt Arch như Manjaro, Antergos, Ninja và nhiều thứ khác. Bạn có thể tải và cài đặt.

2. Cài đặt

Và hướng dẫn sau sẽ hướng dẫn bạn cài đặt qua mạng (sẽ lấy các gói từ lưu trữ từ xa). Bạn sẽ tải file ISO từ trang chủ về và tạo 1 máy ảo trên Virtualbox và bắt đầu như sau.

2.1 Tạo và phân vùng ổ đĩa

Bạn có thể tạo và phân vùng theo nhiều cách, tuy nhiên ở đây tôi dùng cfdisk để phân vùng như sau

root[~]# cfdisk

Sau đó chọn type dos và tiếp tục bạn thực hiện chọn như hình sau

Các bạn cứ tiếp tục thực hiện việc chia tuỳ theo ý thích của bạn. Ở đây tôi chia thành 3 vùng như hình (boot ở /dev/sda1)

Lưu ý bạn cần chọn phân vùng cho swap với kiểu Linux swap như hình

Sau khi chia và chọn kiểu phân vùng thì với mỗi phân vùng t chọn write và nhấn yes để ghi lên các vùng đã tạo.

Sau khi xong chọn quit và ta có thể kiểm tra lại như sau

2.2 Định dạng phân vùng

Tiếp theo ta định dạng các phân vùng vừa tạo

root[~]# mkfs.ext4 /dev/sda1
root[~]# mkfs.ext4 /dev/sda3
root[~]# mkswap /dev/sda2

Phân vùng /dev/sda1 và /dev/sda3 chung định dạng - bạn nhớ định dạng cho tất cả các phân vùng vừa tạo

2.3 Mount ổ đĩa

Để mount các phân vùng trên ta tạo ra một thư mục boot tại /mnt/ và mount tất cả như sau

root[~]# mkdir /mnt/boot
root[~]# mount /dev/sda1 /mnt/boot
root[~]# mount /dev/sda3 /mnt
root[~]# swapon /dev/sda2

2.4 Mirror

Trước khi cài đặt các gói phần mềm bạn nên thêm mirror vào thư mục /etc/pacman.d/mirorlist như sau.

Bạn dùng:

root[~]# nano /etc/pacman.d/mirrorlist

và thêm mirror vào cuối file nội dung sau

## vietnam
Server = http://f.archlinuxvn.org/archlinux/$repo/os/$arch
Server = http://mirror-fpt-telecom.fpt.net/archlinux/$repo/os/$arch

Nhấn Ctrl + X và chọn yes để lưu thay đổi. Tôi dùng mirror của fpt.

Update (2020): có thể bỏ qua bước 2.4 này

2.5 Cấu hình mạng

Đối với Virtualbox bạn có thể bỏ qua bước này. Tuy nhiên việc cấu hình sẽ được thực hiện như sau. Đầu tiên bạn dùng ifconfig để kiểm tra địa chỉ mạng nếu không có mạng bạn có thể dùng

root[~]# dhclient <card mạng>

root[~]# ifconfig <card mạng>

Nếu bạn sử dụng wifi thì có thể dùng lệnh wi-menu để thêm mạng

Update: 2020 (lỗi zsh: command not found: wifi-menu)

Arch linux đã chuyển wifi-menu sang sử dụng iwd và bạn có thể kết nối wifi bằng cách thực hiện lệnh sau:

root[~]# iwctl --passphrase [passphrase] station [device] connect [SSID]

trong đó:

  • passphrase : là mật khẩu của wifi nếu wifi kết nối có sử dụng mật khẩu
  • device: là card mạng (có thể sử dụng lệnh device list sau khi vào mode iwctl)
  • SSID: là tên wifi muốn connect (có thể sử dụng lệnh station [decive] scan sau khi vào mode iwctl để scan danh sách wifi)

xem thêm tại đây

Kiểm tra lại với lệnh ping 8.8.8.8 để xem nối mạng được chưa

2.6 Cài đặt sử dụng Pacstrap

Cài đặt gói vào thư mục gốc

root[~]# pacstrap -i /mnt base base-devel      <- enter
        ...     <- enter sau đó chọn yes

Tùy chọn -i là tránh auto chọn gói.

2.7 Cấu hình hệ thống

Tạo tập tin fstab để tự động mount phân vùng

root[~]# genfstab -U /mnt >> /mnt/etc/fstab

Tùy chọn -U là sử dụng UUISs để định danh nguồn

2.8 Cấu hình timezone

Để bắt đầu các bước tiếp theo bạn vào chế độ tương tác với thư mục gốc đặc biệt của /mnt với các lệnh như hình sau

root[~]# arch-chroot /mnt
root[~]# ln -sf /usr/share/zoneinfo/Asia/Ho_Chi_Minh /etc/localtime

2.9 Chọn ngôn ngữ

Bạn dùng nano và vào thư mục /etc/locale.gen chỉnh sửa sau đó bỏ dấu # phía trước ngôn ngữ muốn chọn hoặc sử dụng lệnh sau (chọn en_US.UTF-8)

root[~]# sed -i s/\#en\_US\.UTF\-8/en\_UTF\-8/g /etc/locale.gen

Sau đó Set như sau để kích hoạt thay đổi

root[~]# echo LANG="en_US.UTF-8" > /etc/locale.conf
root[~]# locale-gen
Generating locales...
    en_US.UTF-8... done
Generating complete.

2.10 Tạo hostname

root[~]# echo "Arch" >> /etc/hostname

2.11 kích hoạt kho multilib

Bạn dùng nano tìm và thay đổi ở vị trí sau của file /etc/pacman.conf

2.11.x Cài đặt kernel, firmware

root[~]# pacman -Syu
root[~]# pacman -S linux linux-firmware
root[~]# mkinitcpio -p linux

2.12 Cài đặt boot loader

root[~]# pacman -S grub-bios

2.13 Tạo Grub và configure Grub

root[~]# grub-install –force /dev/sda

Nếu không thành công, bạn có thể cài đặt với

root[~]# grub-install --target=i386-pc --force /dev/sda

Sau đó

root[~]# grub-mkconfig -o /boot/grub/grub.cfg

2.14 Cài đặt các gói phần mềm quan trọng

Đầu tiên bạn nên update hệ thống với lệnh

root[~]# pacman –Syu

Cài đặt xorg

root[~]# pacman -S xorg xorg-server xorg-xinit xorg-server-utils

Cài đặt net-tools

root[~]# pacman -S net-tools

Cài đặt gvfs. Arch không nhận các thiết bị ngoài như USB, vì vậy cần cài các package này như sau:

root[~]# pacman -S gvfs-mtp gvfs-gphoto2

Cài đặt wicd (sử dụng wireless)

root[~]# pacman -S wicd-gtk

hoặc có thể sử dụng NetworkManager (gnome application)

root[~]# pacman -S networkmanager network-manager-apple

2.15 Thêm tài khoản người dùng

Bạn có thể thêm user và đổi mật khẩu user như sau

root[~]# useradd -m -g users -G wheel -s /bin/bash ctnguyenvn
root[~]# passwd ctnguyenvn
New password:           <- enter password
Retype new password:    <- enter retype password
passwd: password updated sucessfully

2.16 Thay đổi password root

root[~]# passwd root
New password:           <- enter password
Retype new password:    <- enter retype password
passwd: password updated sucessfully

2.17 Start và enable network

Sau đó dùng lệnh sau để bắt đầu và enable mạng khi khởi động

root[~]# systemctl enable dhcpcd

Nếu dùng NetworkManager ở bước 2.14 thì sẽ dùng lệnh

root[~]# systemctl enable NetworkManager

Sau khi xong bạn exit ra ngoài và

root[~]# umount /mnt
root[~]# reboot

reboot lại hệ thống (có thể remove file iso hoặc chọn exiting boot OS)

2.18 Cài đặt môi trường Desktop Environment, Dynamic tiling window manager

Bạn có thể cài đặt DE tùy thích với các lệnh như sau.

Cài đặt xfce

ctnguyenvn[~]$ sudo pacman -S xfce4 xfce4-goodies

Cài đặt gnome

ctnguyenvn[~]$ sudo pacman -S gnome gnome-extra

Cài đặt Cinnamon

ctnguyenvn[~]$ sudo pacman -S cinnamon nemo-fileroller

Cài đặt KDE

ctnguyenvn[~]$ sudo pacman -S plasma

Cài đặt MATE

ctnguyenvn[~]$ sudo pacman -S mate mate-extra

Cài đặt deepin

ctnguyenvn[~]$ sudo pacman -S deepin deepin-extra

Cài đặt LXDE

ctnguyenvn[~]$ sudo pacman -S lxde

Cài đặt i3

ctnguyenvn[~]$ sudo pacman -S i3-wm i3-status dmenu noto-fonts ttf-dejavu ttf-freefont ttf-liberation ttf-droid ttf-inconsolata ttf-roboto

i3 config của tôi tại đây

=> đây tôi cài xfce

ctnguyenvn[~]$ sudo pacman -S xfce4 xfce4-goodies
    ... <- enter

Sau khi cài xong bạn copy file tại /etc/X11/xinit/xinitrc vào thư mục /home/$USER và thêm dòng “exec startxfce4”

ctnguyenvn[~]$ sudo cp /etc/X11/xinit/xinitrc /home/$USER/.xinitrc

vào cuối file và xóa command các dòng có exec ở phía trên như sau

ctnguyenvn[~]$ nano ~/.xinitrc

hoặc bạn tạo 1 file ~/.xinitrc vào thêm vào file dòng lệnh exec startxfce4 vào và lưu lại

Cuối cũng gõ startx để khởi động lại hệ thống.

Bạn có thể cài đặt slim để khởi động vào màn hình desktop mà không cần đăng nhập vào và gõ startx

ctnguyenvn[~]$ sudo pacman -S slim

Sau đó enable slim service

ctnguyenvn[~]$ sudo systemctl enable slim

Kết thúc

Trên đây là những điều cơ bản trong quá trình cài đặt Arch linux. Tất nhiên còn nhiều cách khác cũng như không cần phải cài tất cả những bước trên ta vẫn hoàn thành việc cài đặt distro này. Tuy nhiên tôi nghĩ ta nên cài đầy đủ để trải nghiệm tốt nhất HĐH này. Tôi cũng là một người mới, tôi tự tìm hiểu trên mạng và tự tổng hợp, điều này cũng có nghĩa là sự thiếu xót hoặc sai lầm là điều không thể tránh. Tôi sẽ cập nhật sau.
